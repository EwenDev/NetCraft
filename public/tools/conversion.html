Cours sur la conversion d'IPv4

Qu'est ce qu'une adresse IP ?

Une adresse IP est un identifiant unique attribué à chaque appareil connecté à un réseau informatique utilisant le protocole IP. Elle permet d'identifier de manière unique chaque appareil sur le réseau.

Qu'est ce que la conversion d'adresse IP ?

La conversion d'adresse IP consiste à convertir une adresse IPv4 en binaire ou en héxadécimal. Cela permet de mieux comprendre le fonctionnement des adresses IP et de les manipuler plus facilement.

Comment convertir une adresse IP en binaire ?

Pour convertir une adresse IP en binaire, il suffit de convertir chaque octet de l'adresse IP en binaire. Chaque octet est composé de 8 bits, ce qui permet d'obtenir une adresse IP en binaire de 32 bits. Chaque bit de l'adresse IP correspond à un chiffre binaire, 0 ou 1.

Comment convertir une adresse IP en héxadécimal ?

Pour convertir une adresse IP en héxadécimal, il suffit de convertir chaque octet de l'adresse IP en héxadécimal. Chaque octet est composé de 8 bits, ce qui permet d'obtenir une adresse IP en héxadécimal de 32 bits. L'héxadécimal est une base de numération de 16 chiffres, de 0 à F. Chaque chiffre héxadécimal correspond à 4 bits. Par exemple, le chiffre 0 correspond à 0000 en binaire, le chiffre 1 correspond à 0001 en binaire, le chiffre 2 correspond à 0010 en binaire, etc.

Comment utiliser l'outil de conversion d'adresse IP ?

Pour utiliser l'outil de conversion d'adresse IP, il suffit de saisir l'adresse IP à convertir dans le champ prévu à cet effet. L'outil affichera ensuite le résultat de la conversion en binaire et en héxadécimal.

Attention :
L'adresse IP doit être saisie sous la forme suivante : xxx.xxx.xxx.xxx, où xxx est un nombre compris entre 0 et 255.

Exemple d'adresse IPv4 à convertir :

10.0.30.65 (adresse IP en décimal)
00001010.00000000.00011110.01000001 (adresse IP en binaire)
0A.00.1E.41 (adresse IP en hexadécimal)

178.32.87.19 (adresse IP en décimal)
10110010.00100000.01010111.00010011 (adresse IP en binaire)
B2.20.57.13 (adresse IP en hexadécimal)
